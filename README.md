# Days 6–10 Data Science

A beginner-friendly Data Science learning repository that introduces core statistical concepts through hands-on Jupyter notebooks.

This repository is designed to build a strong foundation in data analysis by exploring how data is cleaned, summarized, interpreted, and analyzed using essential statistical techniques.

---

### Day 6: Comparing Groups
`Lesson6: Comparing Groups.ipynb`

Learn about:
- Mann-Whitney U test
- Welch test
- p-value
- Effect size
- T-statistic
- H0 and H1
- Baselines
- Cuttofs

---

### Day 7: From Samples to the World
`Lesson7: From Samples to the World.ipynb`

Learn about:
- Comparisons
- Patterns and Trends
- Exponential/Log Functions
- Standard Error
- Fetch and Clean

---

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/s326327/Days-1---5-Data-Science.git
```

### Navigate to the Project

```bash
cd Days-6---9-Data-Science
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook and run the cells to follow along with the lessons.

## Learning Outcomes

By completing these notebooks, you will be able to:

- Clean and prepare messy datasets
- Understand measures of central tendency
- Analyze data variability and spread
- Interpret data distributions
- Identify normal and unusual observations
- Build a solid foundation for further Data Science and Machine Learning studies
